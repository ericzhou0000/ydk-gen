# YDK core and bundle stubs

The `sdk` directory contains the `Python`, `C++` and `Go` code for YDK `core` package

```
sdk
├── _docsgen_common # Common documents for C++, Python, Go
├── cpp
│   ├── core # C++ core
│   |      ├── docsgen # Core documentation
│   |      ├── samples # Samples
│   |      ├── src     # Source code
│   |      └── tests   # Core tests
|   ├── gnmi # C++ gNMI
│   |      ├── samples # gNMI samples
│   |      ├── src     # Source code
│   |      └── tests   # gNMI YDK tests
│   ├── install
│   ├── packages
│   ├── samples
│   └── tests # C++ bundle tests
├── go
│   ├── core # Go core
│   |      ├── docsgen # Core and gNMI documentation
│   |      ├── samples # Samples
│   |      ├── tests   # Core & bundle tests
│   |      └── ydk     # Source code
│   ├── gnmi # Go gNMI
│   |      ├── samples # gNMI samples
│   |      ├── tests   # gNMI YDK tests
│   |      └── ydk     # Source code
│   └── packages
├── python
    ├── core # Python core
    |      ├── docsgen # Core and gNMI documentation
    |      ├── samples # Samples
    |      ├── tests   # YDK core and bundle tests
    |      └── ydk     # Source code
    ├── gnmi # Python gNMI
    |      ├── samples # gNMI samples
    |      ├── tests   # YDK gNMI tests
    |      └── ydk     # Source code
    └── packages

```
